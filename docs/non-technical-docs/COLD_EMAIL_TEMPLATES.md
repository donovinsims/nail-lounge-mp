## Sending Guide

| Detail              | Recommendation                                              |
| ------------------- | ----------------------------------------------------------- |
| **Best days**       | Tuesday–Thursday (Thursday peaks)                           |
| **Best times**      | 9–11 AM or 1–3 PM in prospect's local time                  |
| **Avoid**           | Monday mornings, Friday afternoons                          |
| **Sequence length** | 4 emails per set (including breakup)                        |
| **Cadence**         | Day 0 → Day 3 → Day 7–8 → Day 14–21                         |
| **Subject lines**   | 2–4 words, lowercase, no punctuation tricks, no first names |

### Tips for This List

- **Segment before sending:** Single-location → Set A. Multi-location → Set B. Time-pressed signal → Set C.
- **One signal per email.** Pick the strongest Google Maps data point, drop it into the `{{}}` bracket.
- **Local proof wins.** Replace generic proof points with specific local examples once you have them.
- **Reply handling:** "Not interested" → honor it. "Tell me more" → offer 10-min screen share, no pitch deck.
- **Honor the breakup.** If you send Email 4 / C3, do not contact again.
